Instructions to Create a Minimal Web API Application
------------------------------------------------------------------
If you have not already done so, generate the following templates groups:
   Entity/ViewModel Classes
   Data Classes - Entity Framework
   Minimal Web API - Application
   Minimal Web API - Routers and Search Classes

Step 1
Copy the MinimalWebAPI-CreateSolution.cmd file to your desired folder
This .cmd file creates the following folders/projects under this folder
  <|NAMESPACE|>.DataLayer
  <|NAMESPACE|>.EntityLayer
  <|NAMESPACE|>.ViewModelLayer
  <|NAMESPACE|>.MinWebAPI
  PDSC.Common
  PDSC.Common.Web
Run the MinimalWebAPI-CreateSolution.cmd file

Step 2
Open the <|NAMESPACE|>.MinWebAPI.sln file

Step 3
Run your Minimal Web API project
